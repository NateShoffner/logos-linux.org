What is Lo/g/os?

A bloatware-free Arch distro with an installer and UI manager that seeks to succeed where other /g/ made Linux distros have failed. We want to provide more than a simple pre-riced, rebranded distro, and to that end we are creating the riceDB. This will be a simple, portable configuration file manager which will provide users access to a database of rices for any kind of configurable software.

Who is Lo/g/os for?

  • Anyone who needs a quick installation of Arch Linux
  • Anyone who is interested in improving the aesthetics of their desktop
  • Anyone who seeks to learn about how to configure a program
  • Anyone who wants a simple way to manage various configurations for a program
